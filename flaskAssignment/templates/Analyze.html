Analyze the Data

Compare Kickstarter data by using the drop down menus to generate a graph. In the current version, you can specify whether you want the graph to be organized as a histogram of total counts or as a stacked bar plot showing proportions, as well choose a category by which to sort the data.

If you are comparing yearly data, please note that our dataset was collected before 2018 was complete. As a result, the count of 2018 projects will be much lower than others.
